# Archit Mishra Portfolio

Static personal portfolio site built around a Rockefeller-inspired institutional aesthetic.

## Project Setup

- Framework: none
- Routing: single-page anchor navigation
- Styling: plain CSS with custom properties and responsive media queries
- Interactivity: vanilla JavaScript
- Package manager: none

## Files

- `index.html` - page shell and metadata
- `site-content.js` - editable content/data config
- `main.js` - reusable render functions and theme behavior
- `styles.css` - light/dark theme, layout, and responsive styling
- `assets/deco-mark.svg` - architectural visual mark

Open `index.html` directly in a browser, or serve the folder with any static web server.
